# MirrorBreak Matrix v1

**12 Cross-Structural Sub-Dimensions**

## I. COGNITIVE DOMAIN (Struktur Berpikir)

### 1️⃣ Abstraction Bandwidth (AB)
Seberapa jauh seseorang bisa berpikir konseptual, menggeneralisasi, melihat pola besar.

**Skala:**
- 1-3: Concrete (faktual, spesifik, here-and-now)
- 4-6: Hybrid (bisa abstrak tapi butuh konteks konkret)
- 7-10: Highly Abstract (mudah berpikir konseptual, meta-level, sistemik)

---

### 2️⃣ Causal Depth Index (CDI)
Seberapa dalam menganalisis sebab-akibat.

**Skala:**
- 1-3: Surface cause (satu penyebab → satu akibat)
- 4-6: Multi-layer causality (beberapa faktor saling terkait)
- 7-10: Systemic causality (loops, emergent properties, dynamic systems)

---

### 3️⃣ Cognitive Rigidity vs Flexibility (CRF)
Mengukur toleransi ambiguitas dan kemampuan ganti perspektif.

**Skala:**
- 1-3: Binary thinker (hitam/putih, benar/salah)
- 4-6: Contextual thinker (bergantung situasi)
- 7-10: Fluid thinker (mudah pivot, multiple perspectives simultaneously)

---

## II. EMOTIONAL DOMAIN (Arsitektur Emosi)

### 4️⃣ Emotional Granularity (EG)
Kemampuan membedakan emosi secara spesifik.

**Skala:**
- 1-3: Low resolution (senang/sedih/marah saja)
- 4-6: Moderate (tahu variasi dasar)
- 7-10: High precision (bittersweet, anticipatory grief, moral elevation)

---

### 5️⃣ Regulation Strategy Index (RSI)
Cara mengelola emosi:

| Strategy | Description |
|----------|-------------|
| **Suppressive** | Menahan, tidak menunjukkan |
| **Expressive** | Keluarkan secara langsung |
| **Analytical Processing** | Pikirkan dulu, proses kognitif |
| **Avoidant** | Hindari, distraksi |
| **Explosive** | Tumpuk lalu meledak |

*Can be primary + secondary combination*

---

### 6️⃣ Vulnerability Bandwidth (VB)
Kemampuan membuka diri tanpa kehilangan struktur.

**Skala:**
- 1-3: Closed (jarang terbuka, protective)
- 4-5: Selective (terbuka dengan orang tertentu)
- 6-7: Strategic (terbuka dengan purpose)
- 8-10: Fluid authentic (terbuka tanpa kehilangan boundaries)

---

## III. POWER–RELATIONAL DOMAIN

### 7️⃣ Authority Response Pattern (ARP)
Respons terhadap figur otoritas:

| Pattern | Description |
|---------|-------------|
| **Rebellious** | Menolak, melawan, anti-authority |
| **Compliant** | Patuh, mengikuti tanpa question |
| **Strategic Compliance** | Patuh surface, tetap independent |
| **Detached** | Tidak peduli, ignore authority |
| **Dominant-counter** | Kompetisi, ingin jadi authority |

---

### 8️⃣ Recognition Sensitivity (RS)
Seberapa sensitif terhadap diremehkan, diabaikan, tidak dihargai.

**Skala:**
- 1-3: Low (tidak terpengaruh)
- 4-6: Moderate (perhatikan tapi tidak define diri)
- 7-10: High (strong drive untuk diakui, trauma trigger)

---

### 9️⃣ Control Orientation Index (COI)
Mengukur kebutuhan mengatur lingkungan.

**Skala:**
- 1-3: Externalized control (serahkan ke luar, go with flow)
- 4-6: Shared locus (kontrol bersama, negotiable)
- 7-10: Internal locus (perlu kontrol, predictability)

---

## IV. ADAPTIVE CORE DOMAIN

### 🔟 Core Fear Vector (CFV)
Kategori utama ketakutan struktural:

| Fear | Manifestation |
|------|---------------|
| **Rejection** | Ditolak, dikucilkan, tidak diterima |
| **Irrelevance** | Tidak penting, tidak berarti, invisible |
| **Loss of Control** | Ketidakpastian, chaos, helplessness |
| **Abandonment** | Ditinggal, kesepian, tidak ada yang peduli |
| **Incompetence** | Tidak mampu, bodoh, gagal |
| **Powerlessness** | Tidak berdaya, victim, tidak ada pilihan |

*Can have primary + secondary fears*

---

### 1️⃣1️⃣ Defense Mechanism Class (DMC)
Strategi pertahanan dominan:

| Mechanism | Description |
|-----------|-------------|
| **Intellectualization** | Pikirkan supaya tidak rasakan |
| **Humor** | Lelucon sebagai shield |
| **Dominance** | Kuasai situasi sebelum dikuasai |
| **People-pleasing** | Senangkan orang untuk aman |
| **Withdrawal** | Mundur, isolasi, ghosting |
| **Perfectionism** | Kontrol melalui standar tinggi |
| **Hyper-independence** | Tidak butuh siapa-siapa |

*Can be primary + secondary combination*

---

### 1️⃣2️⃣ Adaptive Strength Conversion (ASC)
Bagaimana luka diubah jadi kekuatan:

| Strength | Origin |
|----------|--------|
| **Strategic foresight** | Dari fear of unpredictability |
| **Emotional reading** | Dari hypervigilance terhadap rejection |
| **High discipline** | Dari fear of incompetence |
| **Social calibration** | Dari people-pleasing adaptation |
| **Creative reframing** | Dari intellectualization |
| **System builder** | Dari need for control |

## CROSS-INTERACTION MATRIX

Yang membuat sistem ini kompleks bukan 12 dimensi itu sendiri. Tapi bagaimana mereka **berinteraksi**.

### Contoh Interaksi Kunci

| Pattern | Combination | Output |
|---------|-------------|--------|
| **Structural Strategic Mind** | High AB + High CDI + High CRF | Thinker sistemik yang adaptif |
| **Emotional awareness tanpa ekspresi** | High EG + Low VB + Suppressive RSI | Paham emosi tapi tidak tunjukkan |
| **Dominance adaptation akibat recognition wound** | High RS + Rebellious ARP + High COI | Dominan karena takut diabaikan |
| **Hyper-empathic relational strategist** | High CFV (Rejection) + People-pleasing DMC + High EG | Membaca orang untuk aman |
| **Prestige-driven analytical identity** | High CFV (Irrelevance) + Intellectualization DMC + High AB | Analisis untuk validasi |

### Reading the Matrix

Untuk setiap individu, kita tidak melihat 12 angka terpisah. Kita melihat **konfigurasi** — bagaimana sub-dimensi saling memperkuat atau menegangkan.
