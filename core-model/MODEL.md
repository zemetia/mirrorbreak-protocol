# MBP Core Model

**Sistem profiling struktural berbasis pola empiris**

---

## Metodologi Overview

```
Observe → Hypothesize (Multi-Field) → Stress Test → Affirm/Reject → Refine → Extract Structure
```

| Tahap | Fungsi |
|-------|--------|
| **Observe** | Kumpulkan data: pola respons, narrative, behavioral cues |
| **Hypothesize** | Bangun **multiple fields**, tiap field punya 3+ hipotesis bersaing |
| **Stress Test** | Uji per-field dengan skenario kontradiktif |
| **Affirm/Reject** | Bayesian update: `P(H|E) = P(E|H) × P(H) / P(E)` per field |
| **Refine** | Hipotesis terkuat per field diperhalus iteratif → semakin personal |
| **Extract** | Integrasi cross-field → struktur inti |

## Question Design Methodology ⭐

### Fixed vs Adaptive Ratio per Phase

| Phase | Fixed | Adaptive | Purpose |
|-------|-------|----------|---------|
| **0. Safety** | 100% | 0% | Universal screening, no customization |
| **1. Core** | 70% | 30% | Structural anchors + style-based follow-up |
| **2. Adaptive** | 20% | 80% | Hypothesis-driven questioning |
| **3. Mining** | 0% | 100% | Pure pursuit of adaptation patterns |
| **4. Cross-Val** | 50% | 50% | Fixed contradiction detectors + dynamic stress tests |
| **5-6** | N/A | N/A | Synthesis & debriefing |

### Fixed Question Design: Tension Generators

Fixed questions harus dirancang untuk **sengaja create dimensional tension**.

#### Pattern 1: Somatic-Cognitive Split
> *"Kamu bilang decision-making mu sangat logical. Coba ceritain decision terbesar akhir tahun ini — apa yang tubuhmu rasakan 5 detik sebelum bilang 'yes'?"*

**Targets:** AB (logical claim) × Stress Response (somatic truth)

#### Pattern 2: Temporal Contradiction
> *"Kalau bisa kasih advice ke diri mu 5 tahun lalu, apa yang kamu bilang? ... Terus kenapa advice yang sama nggak kamu apply sekarang?"*

**Targets:** ASC (self-awareness) × COI (consistency over time)

#### Pattern 3: Context Switch
> *"Di kantor, kamu tipe yang gimana? ... Di rumah sama keluarga?"*

**Targets:** RSI (relationship structure) × Compartmentalization detection

#### Pattern 4: Values Hierarchy Test
> *"Kalau disuruh rank: Comfort, Growth, Recognition, Stability — urutannya gimana? ... Terus decision terakhir yang melawan ranking itu kapan?"*

**Targets:** COI (consistency test) — reveal when actions contradict stated values

### Adaptive Question Flow: Hypothesis-Driven

**Jangan pakai template follow-up. Pakai branching logic berdasarkan 12D hypothesis.**

**Example Workflow:**

```
Response: "Aku sih orangnya chill, nggak gampang marah."
    ↓
Analyzer: High VB claim, suppression indicators detected.
    ↓
HypothesisMaker: "Glorified self-image 'stoic/cool', 
                  defense: emotional suppression, 
                  ARP: high-rigid"
    ↓
QuestionMaker generates:
"Kapan terakhir kali kamu bener-bener marah? 
 Bukan annoyed, tapi marah yang sampe pengen break something?"
    ↓
Response: "Hmmm... tahun lalu sih, cuma aku diam aja."
    ↓
Refiner: "Confirmation: Suppression confirmed. 
          Trigger context → legitimacy wound (ASC-related)"
    ↓
Next Question:
"Kenapa kamu pilih diam aja waktu itu? 
 Apa yang bakal terjadi kalau kamu express?"
    ↓
→ Mining adaptation origin
```

### Cross-Validation Question Design

**Teknik "Innocent Mirror":**
> *"Menarik... jadi di situasi A kamu bilang X, tapi di situasi B kamu mention Y. Itu karena context-nya beda, atau memang ada approach yang berbeda untuk situasi berbeda?"*

→ Bukan "kamu contradicted!" tapi "help me understand the nuance"

## Safety-First Architecture ⭐

### Phase 0: Pre-Screening (CRITICAL)

**Hard Contraindications:**
- Active suicidality with plan/intent
- Acute psychotic episode
- Severe dissociative disorder (unmanaged)
- Recent trauma exposure (<3 months) dengan active symptoms

**Soft Contraindications** (proceed with caution):  
- Moderate depression
- History of trauma (stable)
- Personality disorder features (managed)

**Screening Questions (Fixed):**
1. *"Ada nggak hal yang bikin kamu recently feel overwhelmed atau stuck?"* (General distress)
2. *"Kalau ada memory atau topik yang muncul dan bikin kamu uncomfortable, kamu punya coping mechanism yang biasanya work?"* (Regulation check)
3. *"Saat ini, ada nggak pikiran untuk harm yourself atau others?"* (Safety check)

### Phase 6: Debriefing Protocol

**Mandatory Components:**
1. **Resonance Check**: *"Dari apa yang kita bahas, ada yang feel off atau nggak resonate?"*
2. **Grounding**: Resource anchoring, positive memory activation
3. **Exit Strategy**: Contact info untuk follow-up jika needed
4. **Crisis Protocol**: Emergency contacts if distress emerges post-session

## Struktur Inti

| Komponen | Definisi | Deteksi |
|----------|----------|---------|
| **Core Fear** | Tekanan pembentuk | Pola penghindaran, trigger konsisten |
| **Core Drive** | Energi penggerak | Aktivitas dengan energi tinggi, obsesi |
| **Defense Mechanism** | Cara bertahan | Respons otomatis saat threat |
| **Struktur Kognitif** | Cara berpikir | Depth, flexibility, abstraction |
| **Kualitas Emosi** | Cara merasakan | Granularity, regulation, expression |
| **Gap Persona-Inti** | Jarak publik-privat | Inkonsistensi, exhaustion pattern |
| **Adaptasi → Potensi** | Transformasi luka | Strength yang berasal dari wound |

---

## Prinsip Operasional

1. **Setiap sifat adalah adaptasi** — Tidak ada "jelek", hanya respons terhadap tekanan
2. **Kontradiksi > Konsistensi** — Di sinilah struktur terlihat
3. **Luka dan potensi satu sistem** — Tidak bisa dipisahkan
4. **Netralitas** — Bukan hakim, bukan motivator

---

## Output

Structural Profile terdiri dari:
1. Core Fear (hierarchy)
2. Core Drive (primary + secondary)
3. Defense Mechanism (dominan + secondary)
4. 12-Dimensional Matrix scores
5. Persona-Core Gap mapping
6. Adaptasi-Potensi Chain
7. Confidence intervals per komponen
