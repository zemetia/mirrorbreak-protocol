# MirrorBreak Protocol

**Empirical Pattern Deconstruction Model (EPDM)**

A systematic framework for understanding human behavioral patterns through empirical observation and structural analysis.

---

## ⚠️ EPISTEMOLOGICAL CLARIFICATION

**MirrorBreak Protocol is NOT psychometric assessment. MBP is qualitative structural analysis with uncertainty quantification.**

| BUKAN INI ❌ | TAPI INI ✅ |
|--------------|-------------|
| Tes psikometri (MBTI, Big Five, NEO-PI-R) | Wawancara mendalam + pattern recognition |
| Quantitative measurement — angka sebagai truth | Uncertainty quantification — angka sebagai komunikasi ketidakpastian |
| Valid/invalid assessment | Confidence-based interpretation (0-100%) |
| Stable trait measurement | State-dependent structure exposure |
| Score profile | Pattern map + tension analysis |

**Angka dalam MBP (0-100) adalah ordinal ranking untuk komunikasi, bukan interval measurement untuk komputasi.**

## Core Philosophy

> **Observe → Hypothesize → Stress-test → Affirm/Reject → Refine → Structural Extract**

MirrorBreak is not about labeling personalities. It's about understanding the adaptive systems that drive human behavior.

**Fundamental Insight:** Every observable pattern is an adaptation to formative pressure. The goal is not to categorize but to reconstruct the logic of adaptation.

---

## Key Principles

1. **Humans are Adaptive Systems** — Not fixed types. What we observe are: adaptations, compensations, defense structures, energy directions.

2. **Every Pattern is an Adaptation** — There's no "bad" trait. Only adaptations to specific pressures.
   - Excessive control → adaptation to uncertainty
   - People pleasing → adaptation to relational threat

3. **Contradiction is More Informative than Consistency** — The system weights higher on: energy shifts, value tensions, narrative inconsistencies. That's where the core structure reveals itself.

4. **12D Matrix as Tension Network** — 12 interpretive lenses intentionally arranged to contradict each other, exposing gaps between claimed identity and operating structure.

5. **Cultural Context Matters** — Behavior is interpreted within its meaning system. What looks like "suppression" in one culture may be "sabar" (virtue) in another.

### Phase 0: Safety Screening

Empathy mapping, adaptation archaeology, value tension exploration.

### Phase 2: Adaptive Probing
Test hypothesis precision, probe contradiction tolerance, validate/eliminate leads.

Mine adaptation patterns (not trauma events), identify adaptive strength conversions, extract structural signatures.

### Phase 4: Cross-Validation
Apply 12D Matrix tension network, detect persona cracks via cross-dimension contradictions, stress-test hypotheses.

### Phase 5: Structural Synthesis
Integrate field analyses, resolve contradictions (or classify as Type A/B/C integration failure), generate final profile with confidence levels.

### Phase 6: Debriefing
Subjective resonance check, responsible delivery, self-awareness tools, crisis protocols.

## The 12D Matrix: Persona-Breaking Grid

**12D is NOT a measuring instrument. It is a persona-breaking grid** — interpretive lenses that expose structural tensions when they contradict each other.

### Domain Kognitif
- **AB (Abstraction Bandwidth)**: Concrete (0) ↔ Abstract (100)
- **CDI (Causal Depth Index)**: Surface (0) ↔ Systemic (100)
- **CRF (Cognitive Rigidity/Flexibility)**: Binary (0) ↔ Fluid (100)
- **Processing Style**: Intuitive ↔ Analitis

### Domain Emosional
- **EG (Emotional Granularity)**: Low resolution (0) ↔ High precision (100)
- **RSI (Regulation Strategy Index)**: Suppress ↔ Express
- **VB (Vulnerability Bandwidth)**: Closed (0) ↔ Authentic (100)
- **Stress Response**: Freeze ↔ Fight/Flight

### Domain Relasional
- **ARP (Authority Response Pattern)**: Rebellious ↔ Dominant
- **RS (Recognition Sensitivity)**: Low (0) ↔ High (100)
- **COI (Control Orientation Index)**: External (0) ↔ Internal (100)
- **ASC (Adaptive Strength Conversion)**: Legitimacy test — genuine vs glorified adaptation

### Key Tension Pairs (Persona-Breaking)
- **AB × Stress Response**: Claim "analytical" vs somatic freeze (pre-cognitive overwhelm)
- **EG × VB**: High granularity + low bandwidth = emotional trap
- **RS × ARP**: High sensitivity + dominant response = compensation pattern
- **ASC × Emotional Structure**: Legitimacy test for glorification defense

## Confidence Level: Uncertainty Quantification

**Confidence 0-100% = Seberapa yakin kita dengan interpretasi, bukan seberapa "benar" subjeknya.**

| Confidence | Interpretasi |
|------------|--------------|
| 80-100% | Evidence padat, konsisten, contradictions minimal |
| 60-79% | Evidence cukup, beberapa gaps atau tensions |
| 40-59% | Evidence terbatas, high uncertainty |
| <40% | Insufficient data — butuh lebih banyak probing |

**Key principle:** Confidence rendah = butuh data lebih, bukan "failed".

## Cultural Adaptation Framework

MBP recognizes that behavior must be interpreted within its cultural meaning system.

**Open-Ended Cultural Assessment:**
- Parental/family cultural background
- Current residence community norms
- Religious/spiritual intensity and functional role

**Analysis Approach:**
- Map cultural norms (family origin + current)
- Detect deviation: alignment, rebellion, or code-switching
- Interpret within context: "Is this cultural norm or individual defense?"

**⚠️ Religious Context:** Used only as context, never as explanation. Validate spiritual claims against emotional structure (EG, VB, RSI).
